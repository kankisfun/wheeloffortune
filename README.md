# Wheel of Fortune

This program turns a `.txt` file into an interactive **Wheel of Fortune** game.

Each line in the text file becomes one segment on the wheel.  
When the wheel spins, it randomly selects one of the items.

The wheel can be spun manually with the **space bar** or automatically.

---

## How it works

1. Run the program
2. Select a `.txt` file
3. Each non-empty line becomes a wheel segment
4. Press **Space** to spin the wheel
